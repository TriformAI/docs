## Reporting Issues

If you're experiencing issues:

1. **Check this status page** ‚Äî Is there a known incident?
2. **Check third-party status pages** ‚Äî Dependencies listed above
3. **Try basic troubleshooting:**
   - Refresh page (Cmd+Shift+R / Ctrl+Shift+R)
   - Clear cache and cookies
   - Try incognito/private mode
   - Try different browser
4. **Report via:**
   - [Support](/support/contact-us)
   - Email: support@triform.ai
   - Discord: #support channel

**Please include:**
- What you're trying to do
- Error messages (if any)
- Your Organization ID (if applicable)
- Browser and version
- When it started happening
