## Quick Help

**Before contacting support:**

1. **Search docs** — Press `Cmd+K` / `Ctrl+K` to search
2. **Ask Triton** — Chat with the AI assistant in Builder workspace
3. **Check Discord** — Search for similar questions in [discord.gg/triform](https://discord.gg/triform)
4. **Review logs** — Check execution details in Properties → Executions

## Common Issues

### Can't log in

1. Allow popups for `app.triform.ai`
2. Try incognito mode or clear cookies
3. Check OAuth provider status (Discord, GitHub)
4. For 2FA issues, resolve with your provider first

### Execution failing

1. Check logs in Properties → Executions
2. Verify input payload matches schema
3. Test each component individually
4. Confirm Project Variables are set
5. Ask Triton: _"Why did execution #[ID] fail?"_

### Deployment not working

1. Verify all components are configured and tested
2. Check Project Variables for production
3. Confirm API key has deployment permissions
4. Check for circular dependencies

### API requests failing

1. Verify API key is correct and active
2. Use header format: `Authorization: YOUR_KEY`
3. Set `Content-Type: application/json`
4. Confirm payload matches input schema
5. Check rate limits

## Reporting Issues

### Bugs

**Report via:** Bug icon in app, Discord `#bugs`, or support@triform.ai

**Include:**
- What you were doing
- What you expected vs. what happened
- Steps to reproduce
- Screenshots (if helpful)

### Security Issues

**Report to:** security@triform.ai

**Do NOT** post publicly or exploit the vulnerability.

We acknowledge within 24 hours and reward responsible disclosure.
